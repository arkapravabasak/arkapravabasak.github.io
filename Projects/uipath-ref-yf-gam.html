UiPath REFramework – Yahoo Finance Annual Metrics

Project Description

This automation solution uses the UiPath Robotic Enterprise Framework to extract stock data from Yahoo Finance and calculate annual financial metrics.

The solution demonstrates a scalable enterprise automation architecture with robust error handling and retry mechanisms.

High-level process steps

  1. 1. Read the input excel file containing the Stock Ticker list
  2. 2. Open Yahoo Finance in the browser
  3. 3. Search the Company Stock based on it's Ticker
  4. 4. Navigate to the Historical Data Tab for the Stock
  5. 5. Download the 1Year price data
  6. 6. Calculate: Current Price, Average Price across 1 Year, Dates of Highest and Lowest price with date
  7. 7. Save the details in the output excel file
  8. 8. Save the output file with a corresponding date and time stamp
